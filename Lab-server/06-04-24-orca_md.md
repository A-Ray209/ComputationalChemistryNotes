### 1. orca 进行量子动力学模拟

#### 1.1 生成输入文件

```
! r2SCAN-3c  MD                                  # 计算命令
%md
  Timestep 0.5_fs                                # 时间步长 (fs)
  run 200                                        # 总步数
  Initvel 300.0_K                                # 温度 (K)
  thermostat berendsen 300_K Timecon 10.0_fs     # Nose-Hoover 温控器
  Dump position Stride 1 Filename "traj.xyz"
end
%pal nprocs 44                                   # 44 核
     end           
%MaxCore 2327                                    # 2327 M 内存
```
