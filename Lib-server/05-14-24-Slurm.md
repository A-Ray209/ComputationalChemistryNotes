### 1. 安装配置 slurm

#### 1.1 安装 slrum

```
sudo -i
dnf install slurm          # node01（控制端） + node02（服务端）
```
#### 1.2 配置 munge

```
create-munge-key      # node01 生成密钥
scp -r /etc/munge/munge.key node02:/etc/munge/munge.key      # node01 发送密钥到 node02
chown munge:munge /etc/munge/munge.key      # 设置node01 + node 02 

```
